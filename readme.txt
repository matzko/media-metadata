== Description ==

Add meta-data to media objects.

* Media Metadata is developed by [Austin Matzko](http://austinmatzko.com "My blog") ([filosofo](http://profiles.wordpress.org/users/filosofo/ "WordPress.org profile")).
* Media Metadata can be found on [Github](https://github.com/matzko/media-metadata "Github.com URL for Media Metadata").
* Clone it from [git://github.com/matzko/media-metadata.git](git://github.com/matzko/media-metadata.git). Patches (pull requests) welcome!
* The [WordPress Plugin Extend page](http://wordpress.org/extend/plugins/media-metadata/ "WordPress's official plugin page for this plugin").
* Submit support requests [on Github](https://github.com/matzko/media-metadata/issues "The link to the issues page for Media Metadata on Github") or send me an email at austin -at- pressedcode -dot- com.

== Installation ==

1. Upload `media-metadata` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Edit a media item under 'Media' to apply meta-data information.
